# Github Team Remover

## Usage

```ruby
github-team-remover \
  --organization FlatstackSchool \
  --login github-login \
  --password github-password
```

## Install

```bash
bin/bootstrap
```

## Quality tools

* `bin/quality` based on [RuboCop](https://github.com/bbatsov/rubocop)
* `.rubocop.yml` describes active checks

## Develop

`bin/build` checks your specs and runs quality tools
